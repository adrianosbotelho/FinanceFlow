# Política de Segurança

Estamos comprometidos com a segurança deste projeto e levamos a sério qualquer vulnerabilidade relatada.

## Relatando Vulnerabilidades

Se encontrar alguma vulnerabilidade de segurança, por favor, siga as orientações abaixo:

1. Envie um email para [seu-email@example.com] com as informações detalhadas.
2. Não crie uma issue pública para vulnerabilidades de segurança.
3. Aguarde nossa resposta enquanto analisamos e corrigimos o problema.

Agradecemos por contribuir para a segurança deste projeto.
